# Cms9

## Usage
How to use Cms9 plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'cms9'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install cms9
```

And then execute:
```bash
$ rails db:migrate
```
